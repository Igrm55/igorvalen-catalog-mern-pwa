# IgorValen Catalog — MERN + PWA

## Rodar local
1. `npm install`
2. Copie `.env.example` para `.env` e preencha as variáveis (MongoDB Atlas + Cloudinary).
3. `npm run dev`
4. Abra http://localhost:10000

## Deploy no Render
- Build: `npm install`
- Start: `npm start`
- Adicione as variables de ambiente do `.env`.
